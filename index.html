/** * Googleドライブの指定フォルダ内の画像をスキャンし、 * ファイル名（JANコード + アンダースコア + 連番）から * スプレッドシートの「商品」シートへ画像リンクを自動転記するスクリプトです。 * * 【クリーン設計・動作仕様】 * ・スプシ側に該当のJANコードが存在しない画像ファイルは自動的に安全にスルー（無視）します。 * ・スプシ側の該当セルの画像リンクがすでに埋まっている（空欄ではない）場合も、何もしません。 * ・Googleドライブの画像ファイルを移動させたり、名前や説明などの情報を裏で書き換えることはありません。 * ・そのため、スプシのURLを手動で消すだけで、いつでも安全に画像の再転記（やり直し）が可能です。 */ // ========================================== // ⚙️ 初期設定（ご提示のフォルダIDを設定済み） // ========================================== const SOURCE_FOLDER_ID = "1D65ZFKojNHLAE-tTOwxJhdrszIlrgsJg"; // 画像が置かれる対象フォルダのID function syncDriveImagesToSpreadsheet() { const spreadsheet = SpreadsheetApp.getActiveSpreadsheet(); const sheet = spreadsheet.getSheetByName("商品"); if (!sheet) { Logger.log("❌ エラー: 「商品」シートが見つかりませんでした。処理を中断します。"); return; } // 1. スプシの各列のインデックス（列番号）を動的にスキャンして特定 const headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0]; let jan1ColIdx = -1; let jan2ColIdx = -1; const imgColIndices = {}; // { 1: 列番号, 2: 列番号, ... 5: 列番号 } for (let i = 0; i < headers.length; i++) { const h = String(headers[i]).trim().replace(/[\s　]/g, ''); if (h === "JAN1" || h === "jan1" || h === "JANコード1") { jan1ColIdx = i + 1; } else if (h === "JAN2" || h === "jan2" || h === "JANコード2") { jan2ColIdx = i + 1; } // 「画像リンク1」〜「画像リンク5」の列番号を特定 for (let num = 1; num <= 5; num++) { if (h === "画像リンク" + num || h === "画像" + num || h === "image" + num || h === "image_url" + num) { imgColIndices[num] = i + 1; } } } if (jan1ColIdx === -1 && jan2ColIdx === -1) { Logger.log("❌ エラー: JANコードの列（JAN1 または JAN2）を特定できませんでした。"); return; } // 2. Googleドライブ対象フォルダの取得 let sourceFolder; try { sourceFolder = DriveApp.getFolderById(SOURCE_FOLDER_ID); } catch (e) { Logger.log("❌ エラー: フォルダIDが正しくありません: " + e.toString()); return; } // 3. フォルダ内のファイルの取得 const files = sourceFolder.getFiles(); let processedCount = 0; // スプシデータの全取得 const lastRow = sheet.getLastRow(); if (lastRow < 2) { Logger.log("ℹ️ 情報: スプレッドシートに商品データ行が存在しません。"); return; } // getDisplayValues() で取得することで、指数表記(4.57E+12など)の文字化けを100%防ぎ、 // 画面に見えているままのプレーンな「JANコード文字列」をキャッシュします。 const jan1Values = jan1ColIdx !== -1 ? sheet.getRange(2, jan1ColIdx, lastRow - 1, 1).getDisplayValues().map(r => String(r[0]).trim()) : []; const jan2Values = jan2ColIdx !== -1 ? sheet.getRange(2, jan2ColIdx, lastRow - 1, 1).getDisplayValues().map(r => String(r[0]).trim()) : []; // 各画像セルの入力状況を一括キャッシュ const imgValuesCache = {}; for (let num = 1; num <= 5; num++) { const colIdx = imgColIndices[num]; if (colIdx) { imgValuesCache[num] = sheet.getRange(2, colIdx, lastRow - 1, 1).getDisplayValues().map(r => String(r[0]).trim()); } } Logger.log("🔍 スキャン開始: ドライブフォルダ内の全画像ファイルを検証します..."); while (files.hasNext()) { const file = files.next(); const fullName = file.getName(); // フォルダ自体のスキャンはスルー if (file.getMimeType() === MimeType.FOLDER) continue; // ファイル名から拡張子を切り離す（例: "457333660167_01.jpg" -> "457333660167_01"） const lastDotIdx = fullName.lastIndexOf('.'); if (lastDotIdx === -1) { Logger.log("⚠️ スルー（拡張子なし）: " + fullName); continue; } const nameWithoutExt = fullName.substring(0, lastDotIdx); // 【アンダーバールール対応：JANコード + アンダースコア(_) + 01〜05（連番）】 // 例: "457333660167_01"、ハイフン区切りやブレも一部安全許容する正規表現 const match = nameWithoutExt.match(/^(\d+)[-_]?(0?[1-5])$/); if (!match) { Logger.log("⚠️ スルー（ルール不一致）: ファイル名が 'JAN_01' などの命名規則に合致しません: " + fullName); continue; } const fileJan = match[1]; // JANコード部分 const imgNumber = parseInt(match[2], 10); // 画像番号 (1〜5) // スプレッドシート側の対応列が存在するか確認 const targetImgColIdx = imgColIndices[imgNumber]; if (!targetImgColIdx) { Logger.log("⚠️ スルー（列なし）: 「画像リンク" + imgNumber + "」に対応する列がスプシに存在しません: " + fullName); continue; } let isJanFoundInSheet = false; // スプレッドシートの走査 for (let i = 0; i < jan1Values.length; i++) { const rowNum = i + 2; // 2行目から開始 const sJan1 = jan1Values[i]; const sJan2 = jan2Values[i] || ""; // JAN1またはJAN2のいずれかに合致する場合 if (sJan1 === fileJan || sJan2 === fileJan) { isJanFoundInSheet = true; // すでに該当の画像リンク列にURLが入っているかキャッシュ確認 const existingVal = imgValuesCache[imgNumber] ? imgValuesCache[imgNumber][i] : ""; if (existingVal === "") { // 【スルー判定】セルが完全に空欄の時だけ、ドライブURLを安全に書き込み const driveViewUrl = "https://drive.google.com/file/d/" + file.getId() + "/view?usp=drivesdk"; sheet.getRange(rowNum, targetImgColIdx).setValue(driveViewUrl); // キャッシュも同時に更新し、同じ実行内での二重処理を防ぎます if (imgValuesCache[imgNumber]) { imgValuesCache[imgNumber][i] = driveViewUrl; } Logger.log("🟢 転記成功!: 「" + fullName + "」をスプシ " + rowNum + "行目の「画像リンク" + imgNumber + "」に書き込みました。"); processedCount++; } else { // 【スルー判定】すでに値が入力されている箇所は、完全にスルー（上書きをしない） Logger.log("ℹ️ スルー（転記済み）: " + rowNum + "行目の「画像リンク" + imgNumber + "」はすでに入力済みのため何もしません。"); } } } if (!isJanFoundInSheet) { Logger.log("⚠️ スルー（JAN未登録）: 画像ファイル「" + fullName + "」のJAN [" + fileJan + "] はスプシ（JAN1/JAN2列）に存在しません。"); } } Logger.log("========================================="); if (processedCount > 0) { SpreadsheetApp.flush(); Logger.log("🎉 同期完了: 今回新しく " + processedCount + " 件の画像をスプレッドシートに反映しました！"); } else { Logger.log("ℹ️ 同期完了: 新たに転記が必要な画像はありませんでした。"); } Logger.log("========================================="); }
